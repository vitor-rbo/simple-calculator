Learning About Memory Buttons

M+ (Memory Add)

Adds the current value displayed to the memory. Example: User inputs "10" and presses "M+". Memory now contains "10". If user inputs another value (Example: "5"), pressing "M+" updates the memory to the new value ("15").

M- (Memory Subtract)

Subtracts the current value displayed from the memory. Example: If memory had a value of "10" and user inputs "5" and presses "M-", now memory contains "5". Inputing a new value and pressing "M-" updates the memory to the new value.

MC (Memory Clear)

Resets the memory value to "0". Example: Memory had a value of "8". User presses "MC". Memory is cleared and set to "0".

MR (Memory Recall)

Displays the stored memory value on the calculator. Example: User inputs "10" and presses "M+". If user presses "MR" the display will show "10".
